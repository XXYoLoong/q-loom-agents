# v0.3.0

## 目标

- 按用户提供的参考图提升全 3D 控制台视觉质量。
- 从简单 3D 房间升级为精致 Q 版运维工作室：中央长桌、背墙大屏、侧边屏幕机柜、电脑、文件、植物、咖啡、便签、线缆和角色道具。
- 让四个智能体的空间位置和动作更接近参考图：中央敲代码、左侧挥鞭监督、左下放大镜研究、右侧文件比对。
- 保留悬浮状态窗口、点击大屏全屏交互和一键启动能力。

## 变更记录

- 重排 3D 工作坊构图，参考图式布局改为中央敲代码、左侧监督、左下质检、右侧验收。
- 增加左右屏幕机柜、顶部小屏、中央长桌、抽屉、椅子、电脑双屏、阅读桌、咖啡杯、植物、便签、地面线缆。
- 增加 Q 版角色描边、不同发色、脸部表情、眼镜、领带和更明显的任务道具。
- 背景墙 3D 大屏改为控制室仪表盘纹理，包含进度条、网格、折线图和状态模块。
- 移除主界面的网页式启动按钮，改为点击 3D 小人启动闭环，保持主控制台全 3D。
