# v0.2.0

## 目标

- 按外部安装说明安装 PUA Codex skill，但所有落点都必须在 F 盘。
- 增加一键启动脚本，启动后端、前端并打开控制台。
- 将控制台升级为全屏 3D 工作坊房间，而不是网页中嵌入一块静态 3D 图。
- 增加 3D 小人工作动画、悬浮状态窗口、背景墙大屏和点击全屏交互。

## 变更记录

- 新增 `scripts/start-all.ps1`、`scripts/stop-all.ps1` 和根目录 `start-all.bat`。
- 新增 `versions/v0.2.0/INSTALL_PUA_F_DRIVE.md`，记录 PUA skill 的 F 盘安装方式。
- 重建前端 3D 场景为 Q 版数据工坊：房间、地板、墙体、桌椅、电脑、大屏、文件、放大镜、监督鞭子等均用 Three.js 建模。
- 鼠标悬浮小人时显示正在执行的任务窗口。
- 点击背景大屏可打开全屏状态与结果面板。
